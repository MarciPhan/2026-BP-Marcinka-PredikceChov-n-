Advanced API: Action Triggers

Vzdálené ovládání vašeho bota přes REST API. Metricord není jen o datech, je o akci.

1. Odesílání zpráv (Webhooks 2.0)

POST /api/v1/guild/{id}/message

Request Body Schema:

{
  "channel_id": "string",
  "content": "string",
  "embed": {
    "title": "string",
    "description": "string",
    "color": "integer (hex)",
    "fields": [
      { "name": "label", "value": "text", "inline": true }
    ]
  }
}

Příklad odpovědi (201 Created):

{
  "status": "success",
  "message_id": "123456789012345678",
  "channel_id": "999888777"
}

2. Moderační zásahy přes API

POST /api/v1/guild/{id}/member/{uid}/timeout

Parametry: duration (v minutách), reason (nepovinné).

Můžete automatizovat tresty na základě externích signálů.

  • POST /api/v1/guild/{id}/member/{uid}/timeout: Uloží uživateli "time-out".
  • POST /api/v1/guild/{id}/member/{uid}/kick: Vyhodí uživatele ze serveru.

3. Správa rolí

Synchronizujte role s vaším webem nebo e-shopem.

PUT /api/v1/guild/{id}/member/{uid}/roles
{
  "add": ["12345"],
  "remove": ["67890"]
}
⚠️ Bezpečnostní upozornění: Tyto endpointy vyžadují API klíč s oprávněním WRITE_ACTIONS. Nikdy tyto akce nepovolujte pro veřejné tokeny.
