Skóre bezpečnosti (Security Score)

Skóre bezpečnosti je naše proprietární metrika, která kvantifikuje "zdraví" a odolnost vašeho serveru. Je počítáno v reálném čase na základě dat z Discord API a vaší aktivity.

Doporučená hodnota Zdravý komunitní server by měl usilovat o skóre nad 85 bodů. Hodnota 100/100 je dosažitelná, ale vyžaduje perfektní konfiguraci.

Detailní rozpad bodování (0-100)

Celkových 100 bodů je složeno ze 4 hlavních pilířů. Zde je přesný návod, jak získat maximum bodů.

1. Moderátorské pokrytí (Max 25 bodů)

Tento faktor měří, zda máte dostatek "očí" na počet aktivních členů.

  • Logika: Min(25, (Počet Modů / Aktivní Členové) * Koeficient)
  • Ideální stav: 1 aktivní moderátor na každých 500 aktivních členů (DAU).
  • Penalizace: Pokud máte 0 moderátorů, získáte 0 bodů. Bot nepočítá Admina jako moderátora, pokud nemá explicitní roli.

2. Konfigurace Serveru (Max 25 bodů)

Technické zabezpečnení serveru proti botům a spamu.

Nastavení Body Doporučení
Verification Level 10 bodů Alespoň "Low" (vyžaduje ověřený email). "High" (10 minut čekání) je ideální.
Explicit Content Filter 10 bodů Musí být zapnuto skenování médií ("Scan media from all members").
2FA pro Moderátory 5 bodů Zapnutá povinnost 2FA pro role s administrátorskými právy.

3. Zapojení a "Živost" (Max 25 bodů)

Bezpečnost není jen o zámcích, ale i o komunitě. Mrtvý server je náchylnější k převzetí.

  • Měřeno pomocí Stickiness: Poměr DAU / MAU.
  • > 20% Stickiness: Plných 25 bodů. (Každý 5. člen se vrací denně)
  • < 5% Stickiness: 5 bodů. (Server je neaktivní)

4. Historie Moderace (Max 25 bodů)

Provádí moderátoři svou práci? Bot sleduje Audit Log za posledních 30 dní.

  • Aktivní reakce: Zaznamenali jsme bany, kicky nebo timeouty?
  • Plných 25 bodů: Alespoň 5 moderátorských akcí za měsíc (ukazuje, že systém funguje).
  • Bonus: Používání příkazů /warn nebo /timeout.

Doporučení pro zlepšení

V dashboardu pod grafem skóre naleznete sekci Doporučení. Ta dynamicky analyzuje, kde ztrácíte body: 

Příklad doporučení:
"Zvyšte úroveň ověření (Verification Level) v nastavení Discordu."
-> Důvod: Váš server je nastaven na 'None', což umožňuje nájezdy botů. Zisk: +10 bodů.
