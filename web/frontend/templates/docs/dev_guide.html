Developer Guide: Rozšiřování Metricord

Chcete přidat vlastní metriku nebo nový AI model? Tento průvodce vás provede architekturou modulů.

1. Struktura Analytics Modulu

Všechny výpočty metrik probíhají v bot/commands/activity.py a jsou agregovány v bot/commands/stats_hll.py.

Přidání nové metriky (Kroky):

  1. Definujte nový klíč v Redisu (např. stats:custom:{gid}).
  2. Implementujte listener v bot/main.py nebo jako nový Cog.
  3. Přidejte API endpoint do web/backend/main.py pro zobrazení v dashboardu.

2. Vlastní AI Modely

Predikční engine v web/backend/utils.py je navržen jako modulární. Můžete přidat vlastní funkci predict_X, která bere data z Redisu a vrací pravděpodobnost.

# Příklad nového prediktoru

3. Frontend komponenty

Dashboard používá Chart.js a Tailwind-like Vanilla CSS. Pokud přidáváte nový graf, stačí vytvořit nový <canvas> a inicializovat jej v dashboard.html pomocí dat z vašeho nového API endpointu.

Důležité: Vždy zachovávejte asynchronní povahu kódu (async/await) kvůli vysoké propustnosti bota.
