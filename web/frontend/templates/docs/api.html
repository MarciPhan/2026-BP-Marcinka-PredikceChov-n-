API Reference

Metricord poskytuje RESTful rozhraní pro integraci s dalšími systémy. Většina endpointů vyžaduje autentizaci pomocí sezení (session) nebo API klíče.

Base URL: https://dashboard.metricord.app/api/v1

1. Autentizace (Auth API)

Slouží pro přihlašování uživatelů bez nutnosti Discord OAuth2 (např. přes firemní e-mail).

POST
/auth/request-otp

Zašle jednorázový kód (OTP) na zadaný e-mail.

Payload: { "email": "user@domain.com" }
Response: 200 OK { "success": true }
POST
/auth/verify-otp

Ověří kód a vytvoří aktivní sezení.

Payload: { "email": "user@domain.com", "otp": "123456" }
Response: 303 Redirect -> /select-server

2. Správa Discourse (Discourse API)

Umožňuje propojení s diskuzními fóry typu Discourse.

POST
/discourse/add

Přidá nové fórum k účtu uživatele.

Payload: { "url": "https://forum.example.com", "api_key": "...", "api_user": "system" }
Response: 200 OK { "success": true, "guild_id": "-123456" }
POST
/discourse/sync

Vynutí okamžitou synchronizaci dat z fóra do Metricord databáze.

Payload: { "guild_id": "-123456" }
Response: 200 OK { "success": true }

3. Administrace systému (Admin API)

Dostupné pouze pro uživatele s rolí admin.

POST
/admin/config/bot-token

Aktualizuje Discord Bot Token bez nutnosti restartu celého serveru.

Payload: { "token": "MTI..." }
Response: 200 OK
