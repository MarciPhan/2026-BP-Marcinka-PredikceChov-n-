# Brand & Vizuální identita

Metricord není jen nástroj, je to značka. Zde najdete specifikace našeho vizuálního jazyka a designového systému.

## 1. Barevná paleta (Glassmorphism Dark)

| Role | Barva | Hex | Náhled |
| :--- | :--- | :--- | :--- |
| **Background** | Deep Slate | `#0f172a` | ⬛ |
| **Primary** | Electric Violet | `#8b5cf6` | 🟪 |
| **Secondary** | Azure Blue | `#3b82f6` | 🟦 |
| **Glass Border** | White (0.1 Alpha) | `rgba(255,255,255,0.1)` | ◽ |

## 2. Typografie

Používáme moderní bezpatková písma (**Inter / Outfit**) pro maximální čitelnost v datových tabulkách i dlouhých textech.

- **Nadpisy:** `font-weight: 800; letter-spacing: -0.5px;`
- **Kód:** `Fira Code` s podporou ligatur pro matematické operátory.

## 3. Logo a Ikony

Naše logo symbolizuje propojení (**Grit**) a růst (**Apex**). Používáme ikony z rodiny **Lucide** pro konzistentní vizuální styl napříč dashboardem.

::: info Designový princip
"Data first, UI second." Design nesmí nikdy odvádět pozornost od klíčových metrik, musí je pouze elegantně rámovat.
:::
