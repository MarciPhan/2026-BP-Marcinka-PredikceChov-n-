# Podpora Metricord

Jsme tu pro vás 24/7. Vyberte si způsob, který vám nejvíce vyhovuje a my se postaráme o zbytek.

## Kanály podpory

::: tip Komunitní Chat (Discord)
Nejrychlejší způsob, jak získat pomoc. Připojte se na náš oficiální server a zeptejte se v kanálu `#support`.
[**Vstoupit na Discord**](https://discord.gg/metricord)
:::

::: info Emailová Podpora
Máte komplexní problém, našli jste bug nebo máte obchodní dotaz? Napište nám na:
`support@metricord.app` (ozveme se do 24 hodin).
:::

::: info Dokumentace
Projděte si naše detailní návody a [FAQ](/faq). Řešení vašeho problému tam možná už čeká.
:::

## Rychlé otázky

**1. Je tento bot opravdu zdarma?**
Ano, základní verze bota i dashboardu je zdarma. Projekt je vyvíjen jako open-source/hobby.

**2. Co se děje s mými daty?**
Vaše data jsou bezpečně uložena v Redisu a používána pouze pro generování statistik pro váš server. Nikdy je neprodáváme.

**3. Mohu navrhnout novou funkci?**
Určitě! Na našem Discordu máme kanál `#návrhy`. Většinu funkcí implementujeme právě na základě přání komunity.
